# Blog API (Django + Django Ninja)

REST API для блога, построенный на :contentReference[oaicite:0]{index=0} и :contentReference[oaicite:1]{index=1} с использованием :contentReference[oaicite:2]{index=2} и контейнеризации через :contentReference[oaicite:3]{index=3}.

---

## 🚀 Возможности

- Регистрация и аутентификация пользователей через токены
- Создание / редактирование / удаление постов
- Категории постов
- Комментарии к постам
- Авторизация через Bearer Token
- REST API через Django Ninja
- Админ-панель Django

---

## 🧱 Стек технологий

- Python 3.12
- Django 4.2
- Django Ninja 1.2.2
- PostgreSQL 15
- Docker & Docker Compose
- Pytest

---

## 📦 Установка и запуск


```bash
docker compose up --build
```
